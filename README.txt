# Java Scientific Calculator

A Java  calculator built in **Java Swing / AWT** supporting both basic arithmetic and scientific operations, with a modern rounded-button UI.

-----------------------------------------------------------------


## Features


- Basic arithmetic: `+`, `−`, `×`, `÷`, `%`  
- Scientific functions: `sqrt`, `log`, `ln`, `square`, `n!`  
- Parentheses for grouping  
- Decimal numbers  
- Custom-styled UI: rounded buttons and input field  


------------------------------------------------------------------


## Project Structure

 Calculator.java # Main class: GUI, parsing, evaluation
 RoundedButton.java # Custom JButton subclass with rounded corners
 RoundedBorder.java # Custom border for rounded corners on text field

------------------------------------------------------------------


## Requirements: 

-Java JDK 8 or newer

-----------------------------------------------------------------

## To run locally

```bash
git clone https://github.com/Taygeta-gl/Calculator.git
cd Calculator
javac *.java
java Calculator
